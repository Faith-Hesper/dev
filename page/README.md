# Vue 3 + Vite

## 使用方法

1. server 文件夹作后端
2. vue3-demo 为前端
3. 后端使用
   1. npm install
   2. cd server
   3. npm run dev
4. 前端使用
   1. npm install
   2. npm run dev

- LOCATION_C 参考位置
- O_TIME 发震时刻
